## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/space-tourism.git
```

Go to the project directory

```bash
  cd space-tourism
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
